# 常见问题 (FAQ)

本章节汇总了使用 Xinbot 过程中可能遇到的常见疑问。

## Q1: 一定需要正版账号吗？
**不需要**。非正版环境下也可以正常运行。
开启 `onlineMode` 可使用微软正版账号，这能提升在某些服务器的兼容性与可信度。

## Q2: 我可以编写自己的插件吗？
**可以**。Xinbot 提供了完整的类 Bukkit 事件系统和命令系统。
详细步骤请参考 [插件开发指南](../reference/plugin-development)。

## Q3: 如何管理多个机器人？
你可以使用 [xinManager](https://github.com/huangdihd/xinManager) 来实现多机器人的集中管理。

## Q4: 如何处理机器人的移动和地形感知？
我们开发了 [MovementSync](https://github.com/huangdihd/movementsync) 插件，用于实现同步移动与基础的地形管理。
