# Plugin Development Reference

Welcome to the Xinbot Plugin Development Reference. Xinbot features a highly decoupled, plugin-based architecture that allows you to extend the bot's functionality with simple Java code.

### 🚀 Core Documentation
- **[Lifecycle](./plugin-lifecycle)**: Setup your project and master the `onEnable` hooks.
- **[Event System](./event-system)**: Listen to 20+ built-in events like chat, login, and player updates.
- **[Command System](./command-system)**: Add custom console commands with tab completion and real-time highlighting.
- **[Packet Handling](./packet-handling)**: Go low-level and handle raw Minecraft protocol packets.

### 💡 Tips
*   **Java Version**: Use Java 17 or newer.
*   **Dependencies**: Maven or Gradle are recommended.
*   **Study Source**: Read the [xinManager](https://github.com/huangdihd/xinManager) source code for real-world examples.
