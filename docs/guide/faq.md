# FAQ

This chapter summarizes common questions asked when using Xinbot.

## Q1: Does Xinbot require a premium account?
**No**. You can use it in non-premium (offline) mode.
Enable `onlineMode` for Microsoft account login, which can improve compatibility and trust in certain servers.

## Q2: Can I write my own plugins?
**Yes**. Xinbot provides a complete Bukkit-style event and command system.
See the [Plugin Development Guide](../reference/plugin-development) for details.

## Q3: How can I manage multiple bots?
You can use [xinManager](https://github.com/huangdihd/xinManager) for centralized management of multiple bots.

## Q4: How to handle movement and terrain awareness?
We've developed [MovementSync](https://github.com/huangdihd/movementsync) plugin to handle synchronous movement and basic terrain management.

## Q5: How to report issues or request features?
Please open an issue on GitHub. Clear steps to reproduce are highly appreciated.
You can also join the official developer QQ group: **434173700**.
